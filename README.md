# **Hướng dẫn cài đặt cho User**
## ***Hướng dẫn cài đặt xampp để kết nối với cơ sở dữ liệu***
1. Truy cập https://www.apachefriends.org/download.html
2. Chọn phiên bản phù hợp với hệ điều hành
3. Tiến hành cài đặt
4. Mở XAMPP Control Panel sau khi cài đặt
5. Nhấn Start cho cả hai dịch vụ như hình  
   <p align="center"><img width="659" alt="Step5" src="https://github.com/user-attachments/assets/0010b0c7-bbf4-4dcd-9837-ca71226c1ffa"></p>
6. Truy cập localhost theo đường dẫn http://localhost/phpmyadmin/
7. Chọn vào New để tạo 1 cơ sở dữ liệu mới  
   <p align="center"><img width="243" alt="Step7" src="https://github.com/user-attachments/assets/0e5d0f23-cf79-40d3-82d5-0a21b3e48793"></p>
8. Đặt tên cho cơ sở dữ liệu là `sieu_thi_mini` và nhấn vào Create để tạo  
   <p align="center"><img width="1451" alt="Step8" src="https://github.com/user-attachments/assets/7941747f-3eaf-4ea8-9e24-a82f31a6c5c4"></p>
9. Sau khi tạo, cơ sở dữ liệu sẽ hiện ở thanh bên, chọn vào `sieu_thi_mini`  
   <p align="center"><img width="245" alt="Step9" src="https://github.com/user-attachments/assets/b73fef8b-3223-4714-ad13-d6c372217a40"></p>
10. Chọn import để nhập dữ liệu vào cơ sở dữ liệu  
    <p align="center"><img width="1467" alt="Step10" src="https://github.com/user-attachments/assets/c58cd308-ad78-4a82-b1b8-6bdbc4786afe"></p>
11. Chọn Choose File và tìm đến `sieu_thi_mini.sql` thư mục `database` trong thư mục SieuThiMini_MySQL đã giải nén (Xem hướng dẫn tải SieuThiMini_MySQL ở phần sau), sau đó nhấn vào Import  
    <p align="center"><img width="1215" alt="Step11_1" src="https://github.com/user-attachments/assets/f4a02554-e010-450f-b4b8-011eb856baae"></p>
